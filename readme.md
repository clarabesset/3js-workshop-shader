# Prose workshop

### 1. Clone the Repository

```bash
git clone https://github.com/sbcdevp/prose.git
cd minimal-starter
cd quizz-shader
cd tweakpane-editor
cd scroll-product
```

### 2. Install dependencies

```bash
pnpm i
```

### 3. Start the development server

```bash
pnpm dev
```
### 3. Open in browser

Visit http://localhost:3000
